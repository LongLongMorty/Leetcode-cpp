#### 1. [2413. 最小偶倍数](https://leetcode.cn/problems/smallest-even-multiple/)

辗转相除法解决

```c++
class Solution {
public:
    int smallestEvenMultiple(int n) {
        if(n <= 2){
            return 2;
        }
        int a = 2;
        int b = n;
        while(a != 0){
            int temp = b%a;
            b = a;
            a = temp;
        }
        return 2*n/b;
    }
};
```

#### 2.[1512. 好数对的数目](https://leetcode.cn/problems/number-of-good-pairs/)

这道题暴力解没什么问题，但是可以用组合数的知识做，无非就是在重复数里面选两个数。

涉及到一个技巧：杨辉三角DP求解组合数问题

一句话：**组合数满足递推公式，本质就是动态规划**

**核心递推公式**

从 n 个里选 k 个，只有两种情况：

1. 选了第 n 个 → 剩下从 n−1 里选 k−1
2. 没选第 n 个 → 剩下从 n−1 里选 k

所以：

$C(n,k) = C(n-1,k-1) + C(n-1,k)$

这就是**杨辉三角的本质**。

**DP 状态定义**

- 状态：`dp[n][k] = C(n,k)`
- 初始条件：
  - `dp[n][0] = 1`
  - `dp[n][n] = 1`

dp转移方程就是：`dp[n][k]=dp[n−1][k−1]+dp[n−1][k]`

```c++
c[n][0] = c[n][n] = 1;
for (int k = 1; k < n; k++)
    c[n][k] = c[n-1][k-1] + c[n-1][k];
```

这道题固定了`k=2`，所以也可以直接求

```c++
class Solution {
public:
    int count(int n,int l){
        if(l < n){
            return 0;
        }
        if(l == n){
            return 1;
        }
       int r = 1;
       for(int i = l-1;i <= l;i++){
          r *= i;
       }
    
       return r/2;
    }
    int numIdenticalPairs(vector<int>& nums) {
        unordered_map<int,int> r;
        for(int i = 0;i < nums.size();i++){
            r[nums[i]]++;
        }
        int result = 0;
        for(auto it = r.begin();it != r.end();it++){
            result += count(2,it->second);
        }
     return result;
    }
};
```

#### 3.[709. 转换成小写字母](https://leetcode.cn/problems/to-lower-case/)

```c++
class Solution {
public:
    string toLowerCase(string s) {
        for(int i = 0;i < s.size();i++){
            if(s[i] >= 'A' && s[i] <= 'Z'){
                s[i] = s[i]-'A' + 'a';
            }
        }
        return s;
    }
};
```

#### 4.[867. 转置矩阵](https://leetcode.cn/problems/transpose-matrix/)

```c++
class Solution {
public:
    vector<vector<int>> transpose(vector<vector<int>>& matrix) {
        vector<vector<int>> r(matrix[0].size(), vector<int>(matrix.size(), 0));
        for (int i = 0; i < matrix.size(); i++) {
            for (int j = 0; j < matrix[0].size(); j++) {

                r[j][i] = matrix[i][j];
            }
        }
        return r;
    }
};
```
